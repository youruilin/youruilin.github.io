1.17 わたしは　新しい　洋服が　欲しいです

1.17.1　文法説明

  1. 「名」が　欲しいです：表示愿望时：「名詞1」は　「名詞２」が　欲しいです； （あなたは）　何が　欲しいですか；――新しい　パソコンが　欲しいです；
  2. 「名詞」を　「動詞」たいです：相当于汉语“想～”的意思，“たいです”前面接动词去ます结尾后的形式； 以上这两种句式，陈述句时，第一人称为主语，疑问句时，则第二人称“あなた”为主语，两种情况可通过句子的前后关系判断出来，因此常常省略。 （私は）映画を　みたいです；今日は　お酒を　飲みたくないです； （あなたは）何を　したいですか。　――　何もしたくありません；
  3. 「動詞」ませんか：动词的否定形式“～ません”加上表示疑问的“か”，表面上是一种疑问形式，但是语法功能是表示提议,例如：いっしょに　お茶を　飲みませんか；少し　休みませんか；
  4. 「動詞」ましょう：动词的ます结尾换成“ましょう”，表示提议，ちょっと　休みましょ； そろそろ　行きましょう； いっしょに　お茶を　飲みましょう； “ましょう”的礼貌程度，不如“～ませんか”更高；
  5. 疑问词＋でも：表示在任何情况下，事态都都相同，例如：いつでも　電話を　して　ください； 李さん，何を　食べたいですか。――何でも　いいです；だれでも　分かります；
  6. ね②　[缓和语气]：除了第四课的ね表示确认以外，ね也有舒缓自己心情或想法并传达给对方的用法，例如：“まず　恋人が　欲しいですね”、“まじめで　優しい　人が　いいですね”就属于这种用法； 日本料理で　何が　いちばん　好きですか。――そうですね。やっぱり　お寿司ですね；

1.17.2 表現と言葉の説明

  1. ほしです　和 飲みませんか　[愿望，希望和邀请]：这种直接询问他人愿望的表达方式，对长辈或上级使用时，会显得不太礼貌，对长辈或上级，或不太熟悉的人，一般使用“どうですか”或“～ませんか”；デパートへ　買い物に　行きます。李さんも　いっしょに　どうですか。 ――はい，いきたいです；

  2. *今年中に：10课学习了表示整个范围的场所、时间的～中に的用法； ～中に表示该期间结束之前的意思，一般接在表示未来某一特定时间的词后，如“今日、明日、今週、来週、今年”等后面；

    其中“今日、明日、今年”后续～中に时，读“～じゅうに”； “～週、～月（がつ／げつ）”后续～中に时，读“～ちゅうに”； 今年中に（～じゅうに）　結婚したい　です；来週中に（～ちゅうに）もう　一度　連絡します；

  3. ぜひ：一定，必定，后续“～たい”、“～てください”、“～ましょう”时，是表达强烈愿望或意志的提议形式；お汁粉？　ぜひ　食べたいです；今度　ぜひ　家へ　来てください；

  4. お汁粉：年糕小豆汤；

  5. *初詣：新年第一次参拜，祈祷许愿。

  4. “どうぞ　よるしう　お願いします”：自我介绍时，在说完自己的姓名或公司名之后，接着说.. 相当于汉语的“请多关照”，有时也用省略的说法“どうぞ　よろしく”或“よろしく　お願いします”，但“どうぞ　よろしく　お願いいたします。”是较为礼貌的说法。 其中，如改用“…お願いいたします。”的形式，则是更加礼貌的用法。 对方回答时，一般先说“こちらこそ”。

1.2これは　ほん　です

1.2.1 文法説明

  1. 「これ／それ／あれ」は　「名詞」です：【代词】相当于汉语“这，这个、那，那个”；
  2. だれですか／何ですか：“だれ”比较礼貌的说法是*“どなた”；
  3. 「名詞」の　「名詞」：表示所属：　私のかぎ；田中さん　の　車；
  4. 「この／その／あの」　「名詞」は　「名詞」です：【连体词，连接名词】；
  5. どれ／どの　「名詞」：三个以上的事物的情形下使用的疑问词，修饰名词时用“どの”。

1.2.2表現と言葉の説明

  1. 方　「礼貌语言①」：相当于汉语（这/那/哪）位；
  2. 询问年龄 「礼貌语言②」：“何歳ですか。”或更加礼貌的问法“おいくつですか”。全年龄段适用。不过需要注意的是，日语直接询问孩子的年龄时，一般用“いくつ？”或“何歳？”
  3. どうぞ：用于给或赠与对方物品，或劝对方进餐，相当于汉语的“请”；
  4. 「叹词②」：えっ：对对方说出的话感到意外或吃惊时使用，*わあ：感动或吃惊时发出的声音；
  5. “はい”、“ええ”：ええ是はい较随意的说法、只用于肯定对方的疑问。在别人叫自己的名字，必须使用“はい”做回应，而不能用“ええ”；
  6. 外来词,使用片假名来表示；
  7. どうも用于加强 ありがとうございます。所表示的谢意。但是，在略表谢意是，可以只说“どうも”。

1.3ここは　デパートです

#1.3.1文法説明

  1. 「ここ／そこ／あそこ」は　「名詞」です：指示场所的代词，所表示的位置关系，与「これ／それ／あそれ」相一致；
  2. 「名詞」は　「名[场所]」です：表示名词存在的场所。
  3. 「名詞」は　どこですか。：用于询问存在的场所。
  4. 「名詞」も　「名詞」です。：助词“も”，相当于汉语当中的“也”；
  5. 「名詞」は　「名詞」ですか，「名詞」ですか。：答案 有多种可能，询问其中一种时可重复使用位于“～ですか”；
  6. 「名詞」は　いくらですか。：询问价钱时，用“いくら”。

1.3.2表現と言葉の説明

  1. 1階：量词；

  2. 省略②：[谓语的省略]：会话中的进一步询问，一般省略已经说过一遍的谓语部分；

  3. 礼貌语言 ③：

    a.「こちら／そちら／どちら」原本是表示方向的说法，是「ここ／そこ／あそこ」的礼貌说法；

    b. “*お国は　どちらですか。”、“会社は　どちらですか”：询问来自那个国家时，一般使用国は　どちらですか。国前面+お则是更礼貌的说法，但要注意不是所有的名词前面都可以+お。另外，“会社は　どちらですか。包含两层含义：一种是询问公司所在地，一种是询问公司名称，具体判定要根据上下文；

  4. 缩略词：“パソコン”由“パーソナルコンピューター”简化而来，故称为缩略词；

  5. あのう　[搭话　①]：用于搭讪或引起对方注意，另外，再说的难以启齿的事情时、或开启一个新的话题时，以及向别人提出请求时，也可以用“あのう”。

  6. ～ですか　[确认]：用于进一步确认对方所说的信息，句尾读升调。 一百，一千，在日语中表示为百，千，前面不加１，但是一万，一千万，一亿，则需要在前面加上量词1；

1.4部屋に　机と　椅子が　あります

1.4.1文法説明

  1. “あります”、“います”：表示存在：“あります”用于无意志的对象，例如花草，桌子椅子，“います”用于人和动物等具有意志的对象；
    1. 「名詞[场所]」に　「名詞[物/人]」が　あります／います 「名詞[物/人]」は　
    2. 「名詞[场所]」に　あります／います NO3.学习了“～は　どこですか。”这个句型也可以用“～は　どこに　ありますか／いますか”。 哪里有 ~ ？
  2. 「名詞」と　「名詞」[并列]：相当于汉语当中的“和”；
  3. 上／下／前／後ろ／隣／中／外：表示具体位置时，用“名词+の＋上／下／前／後ろ／隣／中／外”，不同于中文里的“桌子上”、“椅子下”，日语中“の”不能省略；
  4. ね　[确认]，读升调；
  5. 疑问词　+　も　＋　「動詞（否定）」：表示全面否定； *教室に　だれも　いません；　　＊冷蔵庫に　何も　ありません；

1.4.2表現と言葉の説明

  1. “上”表示的范围：仅表示垂直上方，日语中“壁のう上に”指墙的垂直上方，屋顶。墙壁的表面用“壁に”：壁に　スイッチ　あります。
  2. ええと：被问及某事，思考该如何回答时的语气词；
  3. “ご家族／ご兄弟／ご両親（Ryousinn）”[礼貌语言④]：说到对方亲属时的礼貌说法+ご；
  4. 兄弟：跟中文不同，日语中的“兄弟”是指同一父母下的兄弟姐妹；
  5. “JR”和“地下鉄”；
  6. 人い暮らし：离开家人一个人生活的状态，被称为“一人暮らし”。
