日本語勉強している·頑張れ！

1.2これは　ほん　です

1.2.1 文法説明

  1. 「これ／それ／あれ」は　「名詞」です：【代词】相当于汉语“这，这个、那，那个”；
  1. だれですか／何ですか：“だれ”比较礼貌的说法是*“どなた”；
  2. 「名詞」の　「名詞」：表示所属：　私のかぎ；田中さん　の　車；
  3. 「この／その／あの」　「名詞」は　「名詞」です：【连体词，连接名词】；
  4. どれ／どの　「名詞」：三个以上的事物的情形下使用的疑问词，修饰名词时用“どの”。

1.2.2表現と言葉の説明

  1. 方　「礼貌语言①」：相当于汉语（这/那/哪）位；
  2. 询问年龄 「礼貌语言②」：“何歳ですか。”或更加礼貌的问法“おいくつですか”。全年龄段适用。不过需要注意的是，日语直接询问孩子的年龄时，一般用“いくつ？”或“何歳？”
  3. どうぞ：用于给或赠与对方物品，或劝对方进餐，相当于汉语的“请”；
  4. 「叹词②」：えっ：对对方说出的话感到意外或吃惊时使用，*わあ：感动或吃惊时发出的声音；
  5. “はい”、“ええ”：ええ是はい较随意的说法、只用于肯定对方的疑问。在别人叫自己的名字，必须使用“はい”做回应，而不能用“ええ”；
  6. 外来词,使用片假名来表示；
  7. どうも用于加强 ありがとうございます。所表示的谢意。但是，在略表谢意是，可以只说“どうも”。

1.1李さんは　中国人　です

1.1.1文法説明

  1. 「名詞」　は　です：　　… 是 … “は”用于提示主题，读作“わ”；

  2. 「名詞」　は　では　ありませこん　：相当于汉语的…不是…，では　ありません　中的“では”，在口语中可以被“じあ”代替。

  3. 疑问句及应答：

    （1）.「名詞」は　「名詞」ですか：　相当于汉语 …是…吗？助词“か”接在句尾表示疑问，不使用“？”，回答时用“はい”或“いいえ”。

    （2）. 应答：回答疑问句时，可以只用“はい”、“いいえ”，也可以在“はい”之后加上“そうです”，“いいえ”之后加上“ちかいます”，即：“はい、そうです。”、“いいえ、ちかいます”。不知道时，用“わかりません”。

  4. 「名詞」の　「名詞」[从属机构、国家][属性]

    助词“の”链接名词和名词，表示前面的名词是后面名词从属的机构、国家或属性。 Attention：日语中无论名词之间是什么关系，一般加“の”，例如：“わたしの　父”，而汉语中表示为我父亲”。

1.1.2表現と言葉の説明

  1. 人称①：（1） “わたし”　“あなた”　“あの　人” 日语中的人称代词，一般来说，第一人称是“わたし”，第二人称是“あなた”，第三人称是“あの　人”。 （２）　～さん：称呼别人是，无论男女，都在其姓名后面+“さん”，“さん”只用于他人的姓名后，不用于自己的姓名后。
  2. 省略①：会话中，多省略第一、第二人称主语，例如： （あなたは）　吉田さんですか。 ――いいえ、（わたしは）　吉田じや　ありません。（わたしは）森です。 小野さんは　会社員ですか。 ――はい、（わたしは）　JC企画の　社員です。
  3. あっ　「叹词①」：句尾的促音，使用：“LTU”或“XTU”组合即可打出来。
  4. “どうぞ　よるしう　お願いします”：自我介绍时，在说完自己的姓名或公司名之后，接着说.. 相当于汉语的“请多关照”，有时也用省略的说法“どうぞ　よろしく”或“よろしく　お願いします”，但“どうぞ　よろしく　お願いいたします。”是较为礼貌的说法。 其中，如改用“…お願いいたします。”的形式，则是更加礼貌的用法。 对方回答时，一般先说“こちらこそ”。
