作成日2025-03-28|更新日2025-03-28
|総文字数:765|読む時間:2分

项目简介

物流项目体系包含四个端：管理后台、用户端、司机端以及快递员端

项目展示

移动端展示

神领物流 - 管理后台（PC端）

项目简介

管理后台：该后台包括了从收到订单开始，通过仓库拣货发货，再通过分拨进行运输、末端配送，然后进行考核、结算等功能

项目迭代日志：

  • v1 - 涉及主流程
    • 后台管理端包含：
      • 基础数据数据
        • 车型管理
        • 车辆管理
        • 司机管理
        • 排班管理
        • 运费管理
      • 营业部管理
        • 快递员管理
        • 快递作业管理
        • 机构作业管理
        • 机构管理
        • 货品类型管理
      • 业务管理
        • 订单管理
        • 运单管理
      • 调度管理
        • 运输任务管理
        • 线路管理

技术栈

vue2+vuex+axios

  • windows 版本 windows 10 企业版
  • 开发工具 vscode
  • node 版本 v14.19.1
  • npm 版本 v6.14.16
